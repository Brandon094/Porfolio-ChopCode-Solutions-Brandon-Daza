Sobre mí

Mi trayectoria en el desarrollo web comenzó en 2021, y desde entonces he estado inmerso en el aprendizaje y creación de soluciones digitales. He trabajado con diversas tecnologías y frameworks, desarrollando tanto proyectos personales como por encargo.

Como fundador de ChopCode Solutions, lidero una empresa dedicada a la innovación tecnológica. Actualmente, ChopCode Solutions abarca dos áreas principales:

  • Desarrollo de Software y Soluciones Digitales:
    Nos enfocamos en la creación de aplicaciones web personalizadas, sistemas empresariales (ERPs), e-commerce y soluciones a medida que impulsan el crecimiento y la eficiencia de nuestros clientes.

  • ChopCode Tech:
    Nuestra división especializada en el mantenimiento de equipos de cómputo, actualizaciones de software, optimización de sistemas y soporte técnico para asegurar el rendimiento óptimo de la infraestructura tecnológica.

Mirando al futuro, planeamos expandirnos con una nueva área enfocada en eCommerce Electrónico, proporcionando soluciones integrales para tiendas online y comercio digital.

Entre mis experiencias más relevantes se encuentran: un sistema ERP para gestión de almacén, dos plataformas e-commerce especializadas, y actualmente me encuentro desarrollando "Ruta Go", una aplicación móvil Android para la venta de puestos de viaje.

Me apasiona enfrentar nuevos desafíos técnicos y busco oportunidades donde pueda aplicar mis conocimientos mientras continúo creciendo profesionalmente.

Mis Proyectos

Ruta Go - Sistema de Transporte Inteligente

App móvil que digitaliza el servicio de transporte tradicional, permitiendo a pasajeros reservar puestos de viaje y a conductores gestionar eficientemente sus operaciones. Desarrollada para resolver problemas de organización y transparencia en rutas establecidas.

  • 🚗 Sistema de rutas y horarios inteligentes
  • 🎫 Reserva digital de puestos de viaje
  • 👥 Gestión dual para pasajeros y conductores
  • 🔥 Base de datos Firebase en tiempo real
  • 📱 Interfaz móvil optimizada
Java Android Studio Firebase Firebase Auth Base de datos en tiempo real Geolocalización

Simplify Biz - ERP Empresarial

Sistema de gestión empresarial desarrollado en Java 17 con interfaz Swing. Gestiona usuarios, productos, ventas e informes con un sistema de roles avanzado y base de datos SQLite.

  • 👥 Gestión de usuarios con sistema de roles
  • 📦 Control de inventario y productos
  • 💰 Módulo de ventas y transacciones
  • 📊 Generación de informes empresariales
  • 💾 Base de datos SQLite configurable
Java 17 Swing SQLite JDBC

E-commerce Amor y Amistad

Tienda virtual especializada en regalos para temporadas festivas, desarrollada con enfoque en performance móvil y experiencia de usuario. Implementa catálogo dinámico cargado desde JSON, carrusel de imágenes y optimización avanzada con formato WebP.

  • 🗃️ Catálogo dinámico cargado desde JSON
  • 🖼️ Carrusel interactivo de productos
  • 🔍 Modal de visualización de imágenes
  • ⏳ Loader animado para mejor UX
  • ⚡ Optimización WebP para imágenes
  • 📱 Diseño 100% responsive
